# macaron
# 마카롱 주문 시스템입니다.
# 마카롱은 100개 한정판매입니다.
# 주문자의 성함을 입력해주시고 주문하실 개수를 입력해주세요
# 주문하신 개수만큼 마카롱 맛을 선택해주시면 주문지가 출력됩니다.
# 이후 주문을 취소하고 다시 주문할지, 주문을 결정하고 포장할지 yes or no를 입력해주세요.
# 주문을 결정하면 각 박스당 4개씩 포장되고 주문자를 불러드립니다.
# 이후 다음주문을 받으며 반복하다 마카롱 100개가 판매되면 프로그램이 종료됩니다.

# 주문 메소드 - 김기수
# 주문서 출력 메소드 - 박건아
# 리셋 메소드 - 송은경
# 포장 메소드 - 김용문
